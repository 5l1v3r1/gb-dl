A python based utility to download courses from infosec4tc.teachable.com and stackskills for personal offline use.

## Usage
To use this script:
```
python gb-dl.py
```
## Requirements
``` 
    beatifulsoup4
    youtube_dl
    wget
    blessings
    requests
    lxml.html
    
```

Tested on ubuntu 16 LTS
